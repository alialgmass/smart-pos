# UI/UX Standards

## Design Principles

- Prioritize clarity over decoration.
- Every screen must have one clear primary action.
- Reduce cognitive load.
- Keep interfaces consistent.

## Spacing

- Use consistent spacing scale.
- Avoid crowded layouts.
- Prefer whitespace for readability.

## Typography

- Maximum 2 font families.
- Maintain clear visual hierarchy.
- Ensure readable font sizes on mobile.

## Colors

- Use semantic colors:
    - Green → success
    - Red → danger
    - Yellow → warning
    - Blue → primary actions

## Forms

- Show validation messages clearly.
- Validate inputs early.
- Keep labels always visible.
- Use proper disabled/loading states.

## Tables & Dashboards

- Important actions visible immediately.
- Use filters/search for large datasets.
- Avoid overwhelming dashboards.

## User Experience

- Every async action needs feedback.
- Avoid dead clicks.
- Use skeleton loaders when possible.
- Minimize user steps to complete tasks.

## Mobile UX

- Thumb-friendly actions.
- Responsive layouts.
- Avoid tiny clickable elements.
